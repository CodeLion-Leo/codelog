## 학습 내용

오늘은 파이썬의 기본 문법에 대해 배웠습니다.
파이썬의 변수 선언과 데이터 타입을 적용하여 간단한 프로그램을 작성해 보았습니다.
프로그램을 실행하여 변수와 데이터 타입의 동작을 확인하고, 기본적인 프로그래밍 개념을 이해할 수 있는 결과를 얻었습니다.

## 코드 예시

```
# 변수 선언 예시

y = '안녕하세요'  # 문자열 변수
```
